NFT Image Copyright Protection System using Image Layer based Creator Authentication

Sogang BaSE Lab · Korea Computer Congress

Verifying creator’s identity by documenting the digital creation process layers and recording this information on the blockchain. This approach aims to safeguard the copyright of image NFTs effectively.
